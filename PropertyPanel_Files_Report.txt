PROPERTY PANEL SYSTEM - FILE LIST & DESCRIPTIONS

--- MAIN CONTROLLERS ---
1. ViewPropertyItems.java
   - The "Engine" of the panel.
   - Dynamically creates UI rows and handles data synchronization between UI and ViewBean.
   - Core method: i(ViewBean) - Saves all UI changes to data.

2. ViewProperty.java
   - The UI container used in the editor sidebar.
   - Manages the view selector spinner and group headers.

3. PropertyActivity.java
   - A standalone Activity for editing properties (full-screen/dialog mode).

--- PROPERTY UI ITEMS (Individual Rows) ---
4. PropertyInputItem.java       - Rows for text/number inputs (ID, Text, Hint, Alpha).
5. PropertyColorItem.java       - Rows for color picking (Text Color, Background Color).
6. PropertySizeItem.java        - Rows for Width and Height presets.
7. PropertySelectorItem.java    - Dropdown menus (Orientation, Text Style, Input Type).
8. PropertyResourceItem.java    - Selectors for Images and Background Resources.
9. PropertyGravityItem.java     - UI for setting Gravity and Layout Gravity.
10. PropertyIndentItem.java     - Detailed controls for Margin and Padding.
11. PropertySwitchSingleLineItem.java - Toggles for Boolean values (Enabled, Checked).
12. PropertyAttributesItem.java - For editing custom XML/HTML attributes.
13. PropertySubheader.java      - Category headers (e.g., "Basic", "Layout").

--- DATA MODELS ---
14. ViewBean.java   - The primary object where all property data is stored.
15. LayoutBean.java - Stores layout-specific data (Width, Height, Margin, Padding).
16. TextBean.java   - Stores text-specific data (Color, Size, Hint).

--- SYSTEM INTEGRATION ---
17. ViewEditorFragment.java - Connects the Property Panel to the Editor Canvas.
18. jC.java                 - Handles global updates (like renaming IDs across the project).
19. cC.java                 - Manages the Undo/Redo history for property changes.
20. oq.java                 - Maps which View types have which Events/Properties.

--- LAYOUT FILES (XML) ---
21. view_property.xml       - Main sidebar layout.
22. property.xml            - Full Activity layout.
23. property_input_item.xml - Layout for text fields.
24. property_color_item.xml - Layout for color fields.
