# Laki Web IDE

Laki Web IDE is a mobile application developed based on the Sketchware UI and its robust features, designed to simplify web development on Android devices.

## Aim and Vision
Our primary goal is to provide a platform that enables simple and efficient web development for everyone. By leveraging a block-based logic system, Laki Web IDE allows users to build websites and web applications without requiring hardcore knowledge of complex web languages (HTML/CSS/JS).

## Key Features
- **Block-Based System**: Highly intuitive modular block system for building CSS and HTML structures.
- **Visual Design**: Drag-and-drop editor inspired by Sketchware Pro, making layout creation visual and easy.
- **Independent Core**: A specialized `core` logic system built for web development.
- **Asset Management**: Easy management of images, fonts, and other web resources.
- **Instant Preview**: View your web projects in real-time as you build them.
- **CSS Property Support**: Comprehensive support for modern CSS properties, including Flexbox, Grid, Animations, and Responsive Queries.

## Project Structure
- `app/src/main/java/laki/webide/core/`: Contains the foundational logic for the block system and editor.
- `app/src/main/java/laki/webide/blockSystem/`: Modular block definitions and rendering logic.
- `app/src/main/res/layout/`: UI layouts for the IDE and custom dialogs.

## How it Works
Laki Web IDE translates visual blocks into clean, standard web code. This abstraction layer removes the steep learning curve of syntax while maintaining the power of professional web development.
